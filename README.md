# To Do List

Простой одностраничный To Do List, разработанный для изучения базовых возможностей Angular.

## О проекте

Проект представляет собой минималистичный менеджер задач с возможностью:

- добавления новых задач;
- удаления отдельных задач;
- отметки задачи как выполненной;
- поиска задач по названию;
- удаления всех задач;
- сохранения списка задач в `localStorage`.

Все данные хранятся локально в браузере и не требуют подключения к серверу.

## Технологии

- Angular
- TypeScript
- SCSS
- HTML5
- LocalStorage API

## Цель проекта

Основная цель проекта — практическое освоение базовых принципов Angular.
